# Installed Plugins

This file documents which plugins and marketplaces should be installed.

## Marketplaces

- **bbrowning-marketplace**: `https://github.com/bbrowning/bbrowning-claude-marketplace`
  - Or local path: `/Users/bbrowning/src/bbrowning-claude-marketplace`

## Plugins

From `bbrowning-marketplace`:
- `claude-builder` - Skills for creating plugins, marketplaces, and skills
- `pr-review` - GitHub pull request review tool

## Installation on New Machine

1. Clone/install marketplaces listed above
2. Use Claude Code to install plugins from those marketplaces
3. Copy `settings.json` and `CLAUDE.md` from this repo
